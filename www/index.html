Server Features

Non-blocking I/O

Single event loop using epoll for efficient handling of multiple concurrent connections without blocking.

HTTP/1.0 Compliant

Full support for HTTP/1.0 protocol with GET, POST, and DELETE methods implementation.

Static File Serving

Efficient static file delivery with configurable document roots and automatic index file serving.

CGI Support

Execute dynamic scripts with proper environment setup, stdin/stdout handling, and process management.

Configuration

Nginx-inspired config files supporting multiple server blocks, locations, and custom error pages.

Security

Built-in security features including request size limits, method restrictions, and access controls.
